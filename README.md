# NetApp Ansible Collections

There are currently 4 NetApp Collections
* ONTAP
* ElementSW (Solidfire)
* CVS for Azure
* CVS for AWS

## Requirements
- ansible version >= 2.9
- netapp-lib version >= 2018.11.13
- requests >= 2.20
- solidfire-sdk-python >= 1.5.0.87

## Installation
```bash
ansible-galaxy collection install netapp.ontap
```
